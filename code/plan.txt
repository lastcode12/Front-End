Task: Create a simple profile card.
Requirements:
    Use semantic HTML
    Card centered on the page
    Includes:
    Profile image (circle)
    Name (heading)
    Short bio (paragraph)
    Button (“Follow”)
Use CSS for:
    Box shadow
    Border radius
    Hover effect on the button
Bonus: Use :hover to slightly lift the card.
